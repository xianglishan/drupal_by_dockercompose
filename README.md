# drupal by docker-compose
---

## how to
```bash
git clone --
cd --
docker-compose up -d --build
```

## apps
- mysql(mariadb)
    - root/password
    - dbname:drupal_db
    - host:mysql
- drupal
    - http://localhost:80/
    - root/pass
